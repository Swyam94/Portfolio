Projects Made

Movie Recommender system

This project is a content-based movie recommendation system built using Python, Pandas, NumPy, and Scikit-learn. It recommends 5 similar movies based on the selected movie using a cosine similarity score computed from features like genres, keywords, cast, and overview. The front-end is developed with Streamlit, providing an interactive and user-friendly interface. Additionally, the system fetches movie posters using the TMDb API, enhancing the visual experience.
